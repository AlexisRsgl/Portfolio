PROJET BOT AFI

PROJET

La projet consiste à ce que l'utilisateur puisse faire des courses sur son mobile et puisse gérer ses différentes listes de courses.


Pour ce faire, nous utiliserons laravel qui est un environnement de développement pour créer l'API qui servira à sécuriser nos requêtes dans la base de données.


Évidemment, nous utiliserons également Ionic pour pouvoir créer notre application sur mobile.


L'utilisation d'Axios sera aussi nécessaire dans le code pour faire appel à notre API laravel.


Nous aurons donc 2 projets différents qui seront séparés pour ensuite pouvoir les relier grâce à Axios. Il y aura donc d'un côté l'API laravel et de l'autre côté les requêtes Axios avec Ionic.

API Laravel

Tout d'abord, nous commençons un dossier pour l'api laravel dans laquelle nous disposerons les fonctionnalités de notre application et les chemins de ces actions appelés routes.


Nous allons créer différents Controllers que nous rendons disponible via une route. Ces routes seront ensuite appelées avec Axios dans le dossier Ionic


Il y a donc 2 Controllers, un pour les produits et l'autre pour les listes de course.


Dans le Controller des produits, nous aurons les fonctionnalités suivantes : Voir tous les produits d'une liste, Création, Modification et Suppression des produits.


Dans le Controller des listes de courses, nous aurons l'affichage de toutes les listes, la création, la modification, la suppression et enfin l'affichage des produits d'une liste.





Axios

Après avoir créer les fonctionnalités avec l'api laravel, nous utilisons axios pour appeller les routes de l'api.


Nous créeons ainsi des requêtes en fonction des fonctionnalités pour pouvoir les réutiliser dans Ionic.


Nous avons donc la vue de toutes les produits d'une liste, la création, la modification et la supression des produits.


Pour ce qu'il s'agit des listes, nous avons l'affichage de toutes les listes, la création, la modification et la suppression.


Nous pouvons maintenant utiliser les requêtes depuis Ionic pour pouvoir obtenir les résultats de la base de données pour notre application mobile.


Et enfin nous sommes libre de faire le front de notre application avec Ionic.
