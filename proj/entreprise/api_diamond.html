PROJET API DIAMOND

PROJET

En étant revendeur de matériels professionnels, nous devons inclure tous les produits des catalogues des fournisseurs.


Pour ce faire ces derniers nous renseignent un lien de leur api nous permettant d'obtenir toutes les données des produits de leur site.


Pour importer les produits sur nos sites, nous passons par un système d'importation via des tableurs excel.


Nous utilisons donc ce lien d'API qui nous permet d'obtenir les données en json.


De ce fait, à l'aide d'excel, nous pouvons les importer et les obtenir sur un tableur excel.


Avec ce tableur excel contenant toutes les données, nous pouvons le modifier à notre guise et l'importer sur nos sites.

Réalisation

Avec l'API Diamond, nous utilisons le logiciel Postman qui nous permet de simuler une connexion à un site et avoir accès à des données spécifiques afficher en json.


Pour ce faire, il suffit d'indiquer le lien API ainsi que les identifiants. Il faut également sélectionner la méthode POST qui permet d'envoyer la requête.


Nous obtenons donc un token qui nous permet par la suite de simuler notre connexion avec les identifiants précèdemment indiqués.


Ce token doit être à la suite de "bearer" dans le champ Authorization. Nous pouvons maintenant envoyer la requête avec la méthode GET afin d'obtenir toutes les données après s'être connecté.


Nous avons enfin toutes les données en json que l'on copie colle dans sublime text et sauvegarder le fichier. Sur excel nous importons les données via ce fichier json et le tableur excel se remplit avec les données.
