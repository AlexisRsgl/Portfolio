PROJET RÉUNION PARENTS PROFS MOBILE

PROJET

Comment simplifier les réunions parents profs des élèves pour qu'ils soient plus organisés et que les parents évitent d'attendre les autres rendez-vous ?


L'idée est de créer une application pour que les parents prennent rendez-vous avec les différents professeurs en fonction de leur décision du moment.


Nous voulons que l'application soit uniquement disposée pour l'affichage mobile.


Pour ce faire, nous utliserons un seul éditeur de code et nous ferons toute l'application en php et html/css.


Nous ferons néanmoins une interface admin pour pouvoir gérer les différents éléments.

Interface admin

Dans l'interface admin, l'utilisateur aura la possibilité de créer différents éléments dans la base de données.


L'admin sera en mesure d'ajouter une classe, une salle, un professeur, une matière, un état civil, un élève et enfin un entretien.


A noter, pour la création d'un professeur, l'état civil et la salle devront être ajoutés en amont. C'est également le cas pour l'élève qui sera attribué à une classe et donc cette dernière devra déjà être présente dans la base de données.


Et enfin, l'admin sera aussi en mesure de créer un entretien dans le cas où il y a eu un problème avec un rendez-vous quelconque ou également dans le cas où un entretien peut être ajouté si certaines circonstances le permette.


Pour créer l'entretien, l'élève, le professeur et la salle devront déjà être intégrés dans la base de données évidemment.





Interface utilisateur (parents)

Pour commencer, le parent s'inscrira sous le nom de l'élève ainsi que sa classe pour que ces derniers soient inscrits dans la base de données.


Il sera redirigé vers une page sur laquelle la liste des professeurs disponibles sera présente.


Le parent d'élève sélectionne donc un des professeurs qu'il souhaite voir en entretien.


Après avoir validé le choix du professeur, une liste des entretiens sera disponible pour pouvoir organiser son temps.


A savoir que les professeurs eux, auront une liste de tous leurs entretiens, avec le nom de l'élève donc.


Et pour finir, nous obtenons le rendu sur mobile.
